# harb

A **H**eap **A**nalyzer for **R**uby. A tool that can be used to analyze Ruby JSON heap dumps that are generated by `ObjectSpace.dump_all`. Should build on OSX and Linux.

#### Building
`make`, or `DEBUG=1 make` for debugging.

#### Usage
`harb <heap_dump_file>`

#### Example
